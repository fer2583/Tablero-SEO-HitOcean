Setup técnico

Cómo enlazar Google Sheets con este tablero.

  1. Convertí el Excel en Google Sheet real.
  2. Abrí Extensiones → Apps Script.
  3. Pegá apps-script/Code.gs.
  4. Publicá como Web App con acceso “Anyone with the link”.
  5. Copiá la URL /exec en config.js, dentro de DATA_URL.
  6. Abrí index.html o subilo a Cloudflare Pages / Netlify.
